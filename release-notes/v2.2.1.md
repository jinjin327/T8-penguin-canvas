# v2.2.1 Source Release

发布日期：2026-06-14

本次为 GitHub 源码版更新，只同步公开源码、文档和功能清单；不打包 Electron，不发布 GitHub Release，不更新自动更新资产。

## 更新内容

- 灵感之源继续收口：艺术风格大师补齐素材右键保存风格、自定义图片上传、分类 / 流派筛选、节点内滚轮浏览、长文字防溢出和输出风格提示词 / 图片链路。
- 新增并完善动漫标签大师节点：参考 comfyui-anima-t8 的懒加载图库思路，支持 Danbooru / Gelbooru 搜索、标签 / 图像输出、自定义分类、导入导出和素材图片右键保存标签。
- 资源库分类管理恢复可用，支持新增、重命名和删除；任务完成提示音支持设置页上传自定义音频，未配置时继续使用默认提示音。
- 画布交互继续修复：顶部快捷栏新增输出素材持久化开关，非选中节点时按 Home 可回到画布中心，文本节点长文缩放不再溢出，整理节点顺序不再颠倒。
- 图像编辑裁切模式新增常用比例与自定义比例，选择比例时默认撑满可用区域，用户可再手动调整裁切框。
- RH 工具箱读取 NodeList 时每次以当前 WebAppId 为准，不再混入上一次应用字段；用户节点隐藏调试映射块，但仍保持输入、参数和固定字段映射完整。
- Codex CLI Agent 简约模式和 Codex 生图工作台补齐上游图片预览、排序、删除和 LLM / IMG 模式边界；模型注意事项新增 `gpt-image-2-vip`。
- 俄罗斯方块主题继续修复节点选中 / 拖动时顶部残影。
- 公开版本同步到 `2.2.1`：`package.json`、`package-lock.json`、`electron/main.cjs`、`backend/src/config.js`、`features.json`、`README.md` 与本 release note 同步。

## 边界说明

- 本次不运行 `npm run dist:release`、`npm run dist`、`npm run release:github` 或 `npm run release:verify`。
- 公开提交继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、Electron 缓存目录、维护者本地制作工具源码和任何私有商业能力 / 分组令牌实现。
- 真实 Grok OAuth 私有调用、New API 分组令牌等私有能力继续通过本机私有扩展 / 加密链路处理，公开仓库只保留安全外壳。
