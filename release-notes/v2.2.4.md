# v2.2.4 Source Release

发布日期：2026-06-15

本次为 GitHub 源码更新：同步公开源码、文档和功能清单，不打包 Electron，不生成安装包 / blockmap / `latest.yml`，也不发布 GitHub Release 自动更新资产。

## 更新内容

- 新增创作台背景层：左侧控制轨可进入画布创作台背景编辑，支持从资源库放置图片、调整透明度、边框、等比缩放、旋转和层级，完成后图片作为节点下方的个性化背景，不阻挡节点编辑与连线。
- RH 工具箱升级为可复用能力中间层：上传素材和输出素材的图像节点在 `Edit` 旁新增 `抠图` 快捷操作，调用 RH 工具箱高清抠图并输出新素材；图像能力预设已预留 `image.cutout`、`image.upscale`、`image.expand`，后续高清化、扩图等能力可复用同一入口。
- RH 图像批处理支持串行队列、默认 3 次尝试、取消和部分成功输出；单张失败不会丢掉已完成结果，失败项会以 `failedItems` 回传。
- AI 去水印节点补齐停止功能，追加隐形处理时可中止长任务；当用户选择 CUDA 但随包 PyTorch 为 CPU-only 时，会回退/提示兼容原因，避免长堆栈误导。
- 上传素材和输出素材节点新增媒体元信息展示：图片显示分辨率，视频显示分辨率和时长，音频显示时长，并保持紧凑不破坏节点 UI。
- ComfyUI 应用制作器继续修复控件手势边界和参数映射稳定性，避免点击标签/控件无响应或把参数改乱。
- 公开版本同步到 `2.2.4`：`package.json`、`package-lock.json`、`vite.config.ts/js`、`electron/main.cjs`、`backend/src/config.js`、`features.json`、`README.md` 与本 release note 同步。
