# 贞贞的无限画布 v2.2.7

本版继续走 Electron 自动更新发布，重点是把长片分镜、素材操作和大画布交互补成更顺手的创作工作流。

## 主要更新

- 新增「导演分镜台」节点：按分镜设置 4-15 秒时长、分辨率、模型、单镜头提示词和多参考素材，并面向 Seedance 2.0 多参考图流程并发生成，单个分镜完成后即可输出。
- 侧边栏支持 `H` 快捷键和主题化按钮隐藏 / 恢复，隐藏后画布控制区自动贴左，给小屏幕更多创作空间。
- 放置栏新增一键清空，并在节点右键菜单中加入「添加到放置栏」，清空后不会因为拖入新节点又恢复旧内容。
- 上传素材 / 输出素材补齐素材删除和复制到剪贴板入口；多图横图预览保持横向比例，操作按钮移到图像上方避免遮挡下载。
- 宫格编辑支持从浏览器外文件夹、画布上传素材和输出素材直接拖入任意格子；宫格剪裁修复 2x2 输出被重复创建多个 4 图输出节点的问题。
- 3D 全景新增「分镜提示板」：可在快照下方拼接黑底白字提示词框，文字高度随输入行数自适应。
- LLM / Vision 长回答路径放宽输出 token 上限，并补齐视频输入 / 高级 provider 相关回归。
- Codex 相关节点和大画布素材收集路径继续做性能收口，降低 Codex 节点加入后拖动画布卡顿。
